## 아이디어 1: 따뜻한 감성 숲과 나침반 (Warm Forest & Compass)

### Design Movement
- **내추럴 미니멀리즘 (Natural Minimalism) & 코지 에스테틱 (Cozy Aesthetic)**
- 자연의 유기적인 곡선과 따뜻한 텍스처를 결합하여 심리적 안정감과 치유의 느낌을 전달합니다.

### Core Principles
- **마음의 평온 (Mindful Calm)**: 여백을 넓게 활용하여 사용자가 차분한 마음으로 테스트에 임할 수 있도록 돕습니다.
- **자연스러운 연결 (Organic Connection)**: 인위적인 격자 무늬 대신 부드러운 유기적 형태(Blob)와 부드러운 그림자를 사용합니다.
- **따뜻한 안내 (Warm Guidance)**: 나침반과 등대 같은 은유적 요소를 통해 내면의 방향을 찾아가는 여정을 시각화합니다.

### Color Philosophy
- 마음을 차분하게 해주는 베이지, 소프트 웜 테라코타, 그리고 숲을 연상시키는 차분한 올리브 그린을 매칭합니다.
- **배경**: 크림 베이지 (`oklch(0.97 0.015 75)`)
- **주요 색상 (Primary)**: 포레스트 그린 (`oklch(0.35 0.06 140)`)
- **강조 색상 (Accent)**: 소프트 테라코타 (`oklch(0.65 0.12 45)`)
- **텍스트**: 차콜 올리브 (`oklch(0.25 0.02 120)`)

### Layout Paradigm
- **비대칭 흐름 (Asymmetric Flow)**: 화면 한쪽에 부드러운 곡선의 일러스트가 배치되고 다른 쪽에 텍스트가 흐르는 자연스러운 레이아웃을 사용합니다. 정형화된 그리드를 피하고 잡지 레이아웃 같은 감성을 전달합니다.

### Signature Elements
- **마음나침반 심볼**: 바늘이 부드럽게 흔들리는 미니멀한 나침반 일러스트.
- **유기적 카드 (Organic Card)**: 모서리가 둥글고 비정형적인 형태를 가진 카드 디자인.
- **종이 질감 (Paper Texture)**: 배경에 아주 미세한 노이즈 패턴을 주어 따뜻한 종이 책을 읽는 듯한 감성을 줍니다.

### Interaction Philosophy
- 버튼 클릭 시 튕기는 듯한 물리적 반응(Spring)을 주어 재미를 느끼게 합니다.
- 페이지 전환 시 책장이 부드럽게 넘어가는 듯한 슬라이드 페이드 효과를 적용합니다.

### Animation
- `--ease-out: cubic-bezier(0.16, 1, 0.3, 1)`을 사용하여 매우 부드럽고 차분한 진입 모션을 제공합니다.
- 문항을 넘길 때 이전 문항이 왼쪽으로 부드럽게 사라지고 새 문항이 오른쪽에서 미끄러지듯 등장합니다.

### Typography System
- **헤드라인**: Serif 서체 (예: Playfair Display 또는 나눔명조 계열)를 사용하여 문학적이고 신뢰감 있는 느낌을 줍니다.
- **본문**: Sans-serif (Pretendard 또는 본고딕 계열)를 사용하여 가독성을 극대화합니다.

## 아이디어 2: 뉴트로 아날로그 일기장 (Retro Analog Journal)

### Design Movement
- **뉴트로 아날로그 (Newtro Analog) & 로파이 에스테틱 (Lo-Fi Aesthetic)**
- 90년대 다이어리, 손글씨 느낌, 격자 노트 등의 감성을 현대적으로 재해석하여 친근하고 개성 있는 느낌을 줍니다.

### Core Principles
- **개인적 기록 (Personal Journaling)**: 나만의 비밀 일기장을 작성하는 듯한 친밀하고 아늑한 경험을 제공합니다.
- **재치와 위트 (Playful Wit)**: MZ세대가 좋아하는 키치하면서도 귀여운 디테일을 곳곳에 배치합니다.
- **직관적 조작 (Tactile Response)**: 아날로그 스위치나 슬라이더 같은 물리적 UI 요소를 현대적으로 구현합니다.

### Color Philosophy
- 톤다운된 파스텔 옐로우와 오렌지, 데님 블루를 조합하여 따뜻하면서도 생기 있는 뉴트로 감성을 연출합니다.
- **배경**: 파스텔 버터 옐로우 (`oklch(0.96 0.04 90)`)
- **주요 색상 (Primary)**: 빈티지 블루 (`oklch(0.45 0.11 240)`)
- **강조 색상 (Accent)**: 번트 오렌지 (`oklch(0.60 0.16 40)`)

### Layout Paradigm
- **다이어리 레이아웃 (Spreading Journal)**: 화면 중앙에 펼쳐진 일기장 형태의 프레임을 고정하고, 그 내부에서 탭을 넘기며 진행하는 레이아웃입니다.

### Signature Elements
- **스탬프 & 스티커**: 결과를 완료하거나 진행할 때 귀여운 가상 스티커가 붙는 효과.
- **격자 패턴 배경 (Grid Pattern)**: 모눈종이 같은 텍스처를 카드 내부에 적용.

### Interaction Philosophy
- 아날로그 버튼을 누를 때 "딸깍"하는 가벼운 햅틱 반응과 사운드(옵션)를 연상시키는 톡톡 튀는 애니메이션.

### Animation
- 팝업이나 모달이 뜰 때 `scale(0.95)`에서 `scale(1)`로 빠르게 튀어오르는 바운스 효과 (`cubic-bezier(0.34, 1.56, 0.64, 1)`)를 적용합니다.

### Typography System
- **헤드라인**: 둥글고 개성 있는 서체(예: 쿠키런체 또는 따뜻한 손글씨체 계열).
- **본문**: 가독성이 높은 둥근 고딕 서체.

## 아이디어 3: 미니멀 소프트 그라디언트 오로라 (Soft Aurora & Light)

### Design Movement
- **소프트 뉴모피즘 (Soft Neumorphism) & 오로라 글래스모피즘 (Glassmorphism)**
- 빛과 투명도, 부드러운 그라디언트를 활용하여 현대적이면서도 몽환적이고 따뜻한 심리 치료실의 조명 느낌을 연출합니다.

### Core Principles
- **내면의 빛 (Inner Light)**: 내면에 숨겨진 자아를 발견한다는 컨셉으로, 은은하게 퍼지는 빛의 그라디언트를 활용합니다.
- **초현실적 투명함 (Translucent Depth)**: 반투명 유리 질감을 사용하여 깊이감 있고 세련된 공간감을 제공합니다.

### Color Philosophy
- 새벽하늘, 노을, 은은한 조명에서 영감을 받은 소프트 오로라 그라디언트를 배경에 은은하게 깔아줍니다.
- **배경**: 매우 연한 라벤더 그레이 (`oklch(0.98 0.01 280)`)에 핑크/골드 그라디언트 블러 배치
- **주요 색상 (Primary)**: 미드나잇 퍼플 (`oklch(0.35 0.08 280)`)
- **강조 색상 (Accent)**: 코랄 핑크 (`oklch(0.75 0.12 25)`)

### Layout Paradigm
- **플로팅 카드 (Floating Card)**: 오로라 빛 배경 위에 반투명한 유리 카드(Glassmorphism)가 공중에 떠 있는 듯한 입체적 레이아웃.

### Signature Elements
- **그라디언트 글로우 (Gradient Glow)**: 마우스 커서나 터치 위치를 따라 부드럽게 움직이는 은은한 빛그림자.
- **프로스트 글래스 (Frosted Glass)**: 뒤 배경의 오로라가 은은하게 비치는 블러 카드.

### Interaction Philosophy
- 모든 요소가 물 흐르듯 부드럽게 움직이며, 사용자의 시선과 마우스 움직임에 따라 미세하게 반응(Parallax)합니다.

### Animation
- 매우 부드럽고 긴 트랜지션 (`duration-500` 이상, `cubic-bezier(0.25, 0.8, 0.25, 1)`)을 사용하여 몽환적인 감성을 극대화합니다.

### Typography System
- **헤드라인**: 세련된 고딕 계열(Pretendard Light/Bold 혼용)로 미니멀하고 정갈한 느낌 전달.
- **본문**: 자간을 넓게 준 깔끔한 산세리프 서체.
